# Glassophite Backend - E-commerce API

Server built with TypeScript, **MongoDB**, Prisma, and Zod validation.

## How to run this project

### Clone the project
    git clone repository-url

### Go to project folder
    cd glassophite_backend

### Install dependencies
    npm install

### Open project to vscode
    code .

### Open terminal & Run project
    npm run dev

### Build project for deploying
    npm run build

### Run build project
    npm run start:prod

## Technology used

- MongoDB (Database)
- Prisma ORM
- Express
- TypeScript
- Zod (Validation)
- JWT (Authentication)
- Bcrypt (Password Hashing)
- Stripe (Payments)
- PipraPay (Payments - self-hosted, open-source)
- DigitalOcean Spaces (File Storage)
